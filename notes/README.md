## 使用方式

- 建議先看 `Chapter 1` 到 `Chapter 5`，把 UNIX 程式設計的共同語言建立起來。
- `Chapter 8` 到 `Chapter 12` 是 process 與 thread 的核心。
- `Chapter 14` 到 `Chapter 19` 偏向 I/O、IPC、terminal 與 network，實作時最常回來查。
- 每章的程式碼都偏重「概念示範」，如果你要，我可以下一輪再補成可直接編譯執行的完整範例專案。
